# Customisations

This directory contains customisation modules for Polarius. These modules allow you to customize various aspects of the application behavior.

## Available Customisations

- **Media.ts** - Customize media handling and processing
- **Lifecycle.ts** - Customize application lifecycle events
- **RoomList.ts** - Customize room list behavior
- **ComponentVisibility.ts** - Control component visibility
- **WidgetVariables.ts** - Customize widget variables
- **WidgetPermissions.ts** - Control widget permissions
- **UserIdentifier.ts** - Customize user identification
- **Directory.ts** - Customize directory behavior
- **ChatExport.ts** - Customize chat export functionality
- **Alias.ts** - Customize alias handling

## Usage

Each customisation module exports specific interfaces that can be implemented to override default behavior. Refer to the individual files for more details on available customisation points.

## Documentation

For more information about customisations, see the [Element Web Module API documentation](https://github.com/element-hq/element-web-module-api).
